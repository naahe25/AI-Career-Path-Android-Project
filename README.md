## 📋 Overview

AI Career Path is a comprehensive cross-platform mobile application designed to provide personalized career guidance and development pathways using artificial intelligence. This project combines native development capabilities with modern mobile frameworks to deliver intelligent career recommendations and progress tracking.

## ✨ Key Features

- **AI-Powered Career Recommendations**: Intelligent algorithms that analyze skills and preferences to suggest suitable career paths
- **Cross-Platform Support**: Native Android (Dart/Flutter) and iOS (Swift) implementations with native performance optimizations
- **Career Path Tracking**: Monitor your career progression with interactive dashboards and analytics
- **Personalized Learning**: Customized learning resources and skill development recommendations
- **Real-time Updates**: Stay informed with latest career opportunities and industry trends
- **User-Friendly Interface**: Intuitive UI designed for seamless user experience

## 🚀 Getting Started

### Prerequisites
- **Flutter SDK**: [Install Flutter](https://flutter.dev/docs/get-started/install)
- **Android SDK**: For Android development
- **Xcode**: For iOS development (macOS)
- **C++ Compiler**: GCC or Clang
- **CMake**: Version 3.16 or higher

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/naahe25/AI-Career-Path-Android-Project.git
   cd AI-Career-Path-Android-Project
   ```

2. **Install Flutter Dependencies**
   ```bash
   flutter pub get
   ```

3. **Build Native Modules**
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   cd ..
   ```

4. **Run the Application**
   ```bash
   # For Android
   flutter run

   # For iOS
   flutter run -d iPhone
   ```
